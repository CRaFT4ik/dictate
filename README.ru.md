# 🎙️ Локальный голосовой ввод

Оффлайн-распознавание речи по хоткею: зажми клавишу (или кнопку мыши), скажи фразу, отпусти — текст распознаётся локально и вставляется в активное окно.

🇬🇧 English: [README.md](./README.md)

---

## ✨ Возможности

- 🔒 **100% оффлайн** — ничего не уходит с вашего ПК
- 🧠 **Две модели на выбор** при первом запуске
  - **Whisper large-v3** — мультиязычная (99 языков)
  - **GigaAM v3 e2e_rnnt** — только русский, ~2× точнее, лучше пунктуация
- ⚡ **Ускорение на CUDA** (CPU тоже работает)
- ⌨️🖱️ **Любой хоткей** — клавиатура, мышь или комбо
- 📋 **Умный буфер обмена** — текст и картинки переживают вставку
- 📦 **Авто-установка** зависимостей при первом запуске — `pip install` руками не нужен

## 🚀 Быстрый старт

```bash
git clone <repo-url>
cd dictate
```

Запуск:

| ОС | Команда |
|---|---|
| 🪟 Windows | `.\dictate.ps1` (от администратора) |
| 🐧 Linux   | `sudo bash dictate.sh` |
| 🍎 macOS   | `bash dictate.sh` |

**При первом запуске:**
1. Зажми желаемое комбо (например `Ctrl + Space`) — сохранится в `voice_key.json`
2. Выбери модель: `[1]` Whisper или `[2]` GigaAM

Выбор запоминается. Зависимости только выбранной модели поставятся автоматически.

> Нужен Python 3.10+ установленный заранее.

**Как работает:** скрипт запуска стартует `client.py`, который при необходимости поднимает `server.py` на `127.0.0.1:7531`. Сервер продолжает работать после закрытия клиента — остановить вручную: `taskkill /F /IM python.exe` (Windows) / `pkill -f server.py` (macOS / Linux).

**Сетевой доступ:** по умолчанию сервер слушает на `0.0.0.0:7531`. Для работы по локальной сети откройте порт:
- **Windows** (PowerShell от администратора): `netsh advfirewall firewall add rule name="Dictate" dir=in action=allow protocol=TCP localport=7531`
- **Linux**: `sudo ufw allow 7531`
- **macOS**: подтвердите запрос брандмауэра при первом запуске сервера на `0.0.0.0`.

## 📱 Android-клиент (опционально)

Зажми перетаскиваемую плавающую кнопку в любой точке экрана, скажи фразу, отпусти — текст вставляется в активное поле ввода через Accessibility paste.

**Настройка:**
1. Отредактируй `android/app/src/main/local.properties` (создай, если файла нет): `server.url=http://<IP-твоего-ПК-в-локальной-сети>:7531`.
2. Собери и установи приложение на устройство в той же Wi-Fi-сети:
   ```bash
   cd android
   ./gradlew installDebug
   ```
3. Запусти приложение и выдай три запрошенных разрешения (микрофон, наложение поверх других окон, специальные возможности).
4. Нажми «Включить плавающую кнопку» или опусти шторку Quick Settings и нажми плитку «Dictate».
5. Долгое нажатие на плавающую кнопку (~200 мс), скажи фразу, отпусти.

Python-сервер на ПК должен быть запущен и доступен по локальной сети. Откройте порт `7531` согласно инструкциям из раздела «Сетевой доступ» выше.

**Смок-тест (эмулятор):** см. `android/scripts/adb_smoke.sh` (или `.ps1`).

## 🎯 Какую модель выбрать

| | Whisper large-v3 | GigaAM v3 e2e_rnnt |
|---|---|---|
| Языки | 99 | только русский |
| WER на русском | ~21% | **~11%** |
| Пунктуация | ✅ | ✅ (заметно лучше, особенно запятые) |
| Размер модели | ~3 ГБ | ~0.8 ГБ |
| Доп. зависимости | мало | тяжёлые (torch + transformers, ~2 ГБ) |

Для русского — **GigaAM**, для остальных языков — **Whisper**.

## ▶️ Использование

Зажми хоткей, скажи фразу, отпусти — распознанный текст вставляется в активное приложение.

**Перенастройка:**
- `python client.py --rebind`  — сменить хоткей
- `python client.py --remodel` — сменить модель

---

## ⚙️ GPU / CPU

Обе модели работают на NVIDIA GPU (по умолчанию) или CPU. Для GPU нужен свежий драйвер NVIDIA — проверь через `nvidia-smi`. Чтобы принудительно использовать CPU — поставь `DEVICE = "cpu"` в `server.py`.

Если `faster-whisper` падает на CUDA/cuDNN, зафиксируй совместимую версию: `pip install --force-reinstall ctranslate2==4.4.0`.

## 🧽 Фильтр галлюцинаций

Whisper иногда выдаёт «спасибо за просмотр» на тишине. `hallucinations.txt` вырезает такие фразы из вывода:

- `какая-то фраза`  — точное совпадение
- `~подпишитесь`    — подстрока
- `^спасибо.*`      — regex
- `# комментарий`   — комментарий

## 📝 Примечания

- 🛡️ Глобальные хуки клавиатуры/мыши требуют admin/root на Windows и Linux.
- 🐧 На Linux Wayland часто блокирует глобальные хуки — X11 сессии работают надёжнее.
- 📋 Буфер обмена сохраняется при вставке (текст и картинки, на всех платформах).
