# 🎙️ Локальный голосовой ввод по хоткею (Embedded Faster-Whisper)

Оффлайн-голосовой ввод, который запускает **embedded-модель распознавания локально на вашем ПК** (без облака): зажми хоткей/кнопку мыши → говори → Faster-Whisper распознаёт речь и **автоматически вставляет** текст в активное окно.

🇬🇧 English version: [README.md](./README.md)

---

## ✨ Что умеет
- 🔒 **Локальное / оффлайн** распознавание (ничего не отправляется в облако)
- ⚡ **Ускорение на GPU (CUDA)** через `faster-whisper` (опционально, CPU тоже работает)
- ⌨️🖱️ **Произвольный хоткей-комбо** (клавиатура + мышь)
- 📋 **Автовставка текста** в активное приложение (буфер обмена + вставка)
- 🎚️ **Два режима**
  - 🧠 **Целиком (по умолчанию):** запись пока зажата клавиша → распознавание после отпускания (лучшее качество)
  - 🚀 **Стриминг:** делит на фразы по паузам/таймеру и вставляет по ходу (быстрее, но меньше контекста)
- 🧹 **Фильтр “галлюцинаций”** через `hallucinations.txt` (точное совпадение / подстрока / regex)

## 🚀 Быстрый старт

### 🧩 Установка
1) Клонировать репозиторий и создать venv
- `git clone <repo-url>`
- `cd <repo-folder>`
- `python -m venv .venv`

2) Активировать venv
- Windows: `.venv\Scripts\activate`
- Linux/macOS: `source .venv/bin/activate`

3) Установить зависимости
- `pip install -U pip`
- `pip install -r requirements.txt`

Пример `requirements.txt`:
- `faster-whisper`
- `sounddevice`
- `numpy`
- `pyperclip`
- `keyboard`
- `mouse`

## 🧠⚡ Установка CUDA (GPU)

Проект умеет работать на NVIDIA GPU через CUDA (поставьте `DEVICE="cuda"`). Нужно:

1) Драйвер NVIDIA
- Установите свежий драйвер для вашей видеокарты.
- Проверьте, что GPU виден:
  - `nvidia-smi`

2) CUDA Toolkit (опционально)
- Часто достаточно только драйвера, потому что многие Python-пакеты приносят CUDA runtime библиотеки с собой.
- Если нужен полный Toolkit, ставьте с сайта NVIDIA:
  - Загрузка CUDA: https://developer.nvidia.com/cuda-downloads
  - Гайд для Windows: https://docs.nvidia.com/cuda/cuda-installation-guide-microsoft-windows/
  - Гайд для Linux: https://docs.nvidia.com/cuda/cuda-installation-guide-linux/

3) Установка Python-зависимостей
- `pip install -r requirements.txt`

Важно: совместимость faster-whisper / CTranslate2 с CUDA
- `faster-whisper` использует `ctranslate2`, и важна совместимость по CUDA/cuDNN.
- Если видите ошибки типа “Could not load cudnn” / “CUDA version mismatch” / “No CUDA device”, попробуйте:
  - Убедитесь, что работает `nvidia-smi` (драйвер установлен)
  - Обновите драйвер NVIDIA
  - Переустановите зависимости в чистом venv
  - Зафиксируйте `ctranslate2` на совместимой версии (пример):
    - `pip install --force-reinstall ctranslate2==4.4.0`

Если с GPU не хочется возиться — переключитесь на CPU:
- Поставьте `DEVICE="cpu"` в `dictate.py`

## ▶️ Запуск
- 🪟 Windows: `dictate.bat` (с правами админа)
- 🐧 Linux: `bash dictate.sh`

При первом запуске зажмите нужную комбинацию хоткея (~1 секунда) — она сохранится в `voice_key.json`.

Переназначить хоткей:
- `python dictate.py --rebind`

---
 
## ⚙️ Настройки (вверху файла `dictate.py`)
- `MODEL_NAME` — имя модели Whisper (например: `large-v3-turbo`)
- `DEVICE` — `cuda` или `cpu`
- `LANGUAGE` — код языка (например: `ru`, `en`, `de`), или `None` для авто-определения
- `STREAMING_MODE` — `False` (лучшее качество) / `True` (вставка во время речи)

## 🧽 Фильтр галлюцинаций
Файл: `hallucinations.txt`
- точное совпадение строки: `какая-то фраза`
- подстрока: `~подпишитесь`
- regex: `^(спасибо|благодарю).*`
- комментарии: строки, начинающиеся с `#`

---

## 📝 Примечания
- 🛡️ Используются глобальные хуки клавиатуры/мыши — иногда нужны права администратора/root.
- 🐧 На Linux работа хоткеев/вставки зависит от X11 vs Wayland (Wayland может блокировать глобальные хуки/инжект ввода).
- 📋 Автовставка использует буфер обмена временно и затем восстанавливает его.
